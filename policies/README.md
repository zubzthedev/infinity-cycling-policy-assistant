# Policies

Drop the club's official governance documents here as Markdown (`.md`) files.

They are loaded once at application startup and cached in memory (see `app/policies.py`,
added in Milestone 3). An administrator can trigger a reload after adding, replacing, or
deleting a file without restarting the application.

Expected documents (see the build specification):

- `constitution.md`
- `disciplinary.md`
- `membership.md`
- `racing.md`
- `code_of_conduct.md`
